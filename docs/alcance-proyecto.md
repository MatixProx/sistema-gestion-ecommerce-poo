# Alcance del Proyecto

## Sistema elegido
Sistema de Gestion de e-commerce.

## Objetivo
Permitir que una tienda en linea administre productos, clientes, pedidos, pagos, inventario y reportes administrativos.

## Incluye
- Registro e inicio de sesion de usuarios.
- Administracion de clientes y administradores.
- Registro, modificacion, consulta y eliminacion de productos.
- Clasificacion de productos por categorias.
- Carrito de compras con calculo de subtotales y total.
- Generacion de pedidos.
- Simulacion del proceso de pago.
- Actualizacion basica de inventario.
- Reportes de ventas y productos con bajo stock.
- Configuracion de repositorio en GitHub.

## No incluye en la primera etapa
- Integracion real con bancos.
- Facturacion electronica oficial.
- Aplicacion movil nativa.
- Recomendaciones con inteligencia artificial.
- Conexion real con empresas de envio.

## Valor empresarial
El sistema ayuda a mejorar el control de ventas, productos e inventario. Tambien facilita la toma de decisiones mediante reportes administrativos y reduce errores manuales en el proceso comercial.
